# Getting Started with Bounding Boxes on your Smartphone

Just clone this project to start. And create Bounding Boxes on your Smartphone in a mobile Web optimised way.

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.

### `npm run build`

Builds the app for production to the `build` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.

The build is minified and the filenames include the hashes.\

## Available functionalities

You can create Bounding Boxes with React on your Smartphone. This Mobile Web UI optimised application uses the Konva React Libary for most of it's functions.

### Create Bounding Boxes for your own images

Just go into the Image Component and change the Domains of the COCO Dataset. Just put your own Domains there and get started.

### Read the JSON Files of the Boundingboxes

Just Save a Bounding Box and look in your lokal Browser Storage. The you finde the JSON Files unter the key "BoundingBox". The Format is X-Value, Y-Value, width and hight.
